This color scheme generator is my first project where I implement an API. 

The api: https://www.thecolorapi.com/docs#schemes-generate-scheme-get

Given some queries and a GET request, it will return a color scheme given the user's inputs. This project helped me practice:
- Using fetch requests in JS
- Practice handling basic API requests
- Implementing API JS code with HTML and CSS frontend skills.
