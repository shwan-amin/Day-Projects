This front-end project is a watchlist type mini-app where you can search for movies via the OMDb API and then add these to your watchlist.

The API: https://www.omdbapi.com/

This project helped me practice:
- Multi-page HTML front-end design
- CSS design
- Using local storage to have data persist across pages
- API Calls, Async functions and await
- Manipulating the DOM via JS
