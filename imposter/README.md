This project was a mini-challenge I set to myself to try recreate the popular word game Imposter, except theres a twist where everyone is given fruits instead of words! Though the game may not be extremely replayable, it still might be just as good as the original. 

This project was made so I could further reinforce skills I learned in JavaScript DOM manipulation, event handling, and making a shallow one-page site with some interesting features.

Specifically I practiced:
- Using fixed, relative and absolute positions
- Creating HTML elements in JS
- Working with setTimeout logic flow
- Creating game logic for a static site
